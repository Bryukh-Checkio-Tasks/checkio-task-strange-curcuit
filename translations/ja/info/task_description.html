ニコラは奇妙な回路基盤を拾った。 それは可能な限り全ての要素がらせん状に縦横でつながっていた。

この回路基盤は正方形のマスでできている。最初の要素は中央の1であり、 あとは昇順に時計回りでらせん状に続いている。 あなたはこの図上で隣接したマスを上下左右に移動可能なので ニコラが2点間のマンハッタン距離を数えるのを手伝うことができる。 例えば1と9の間は2で、24と9の間は1である。

入力: 2つのマスの数値 (integer型)

出力: 2マス間のマンハッタン距離 (integer型)

例: 

{% if interpreter.slug == "js-node" %}
findDistance(1, 9) == 2
findDistance(9, 1) == 2
findDistance(10, 25) == 1
findDistance(5, 9) == 4
{% else %}
find_distance(1, 9) == 2
find_distance(9, 1) == 2
find_distance(10, 25) == 1
find_distance(5, 9) == 4
{% endif %}

どのように使われるか: このタスクは一般的でない座標系を調査するのに役立つ。 この座標系は地形学で時々使用される。 またこのタスクはウラムの螺旋(素数螺旋)と関係がある。 ウラムの螺旋はシンプルな素数の可視化手法であり、 ある二次方程式が非常に多くの素数を生成する傾向があるのをを明らかにした。

事前条件: 0 < first < 1000
0 < second < 1000
first != second
